0052: Copy non-zero Data
Copy data from hl to de while the data is non-zero.
Input
hl Source address
de Destination address
L0052 0052 ret
This entry point is used by the routines at L0302 and L0318.
L0052_0 0053 ld a,(hl) Read data from hl and check if it is zero.
0054 and a
0055 ret z If it is zero, we are done.
0056 ld (de),a If not zero, store it and
0057 inc hl increment the pointers
0058 inc de
0059 jr L0052_0 Loop to the next byte.
