0ca0: Display message
Display the string pointed to by hl terminated by a value >= $7f.
Input
used by the routines at init, L0c77 and L0c88.
displayMessage 0ca0 ld a,(hl) Get a character from the string
0ca1 cp $7f and compare it with $7f.
0ca3 push af Save the flags and
0ca4 and $7f remove bit 7.
0ca6 rst $10 Display character
0ca7 pop af Restore the flags.
0ca8 ret nc If no carry, end of string.
0ca9 inc hl Move the string pointer to the next character
0caa jr displayMessage and loop.
