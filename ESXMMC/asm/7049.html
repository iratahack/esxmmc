1b89: Copy code
Copy 4 bytes of code from $1ffc to the address passed in hl.
Used by the routine at init.
L1b89 1b89 ex de,hl Move the destination address to de.
1b8a ld hl,$1ffc Load hl with the source address.
1b8d ldi Copy 4 bytes.
